## Getting Started

```
.env
NEXT_PUBLIC_API_URL=http://localhost:3000/api/0326e7fc-6f4d-427b-bbcc-04dcc189095c
```
First, run the development server:

```bash
npx prisma generate
npx prisma db push

npm run install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
